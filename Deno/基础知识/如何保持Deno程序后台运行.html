# 用PM2运行Deno脚本

PM2可以运行Node.js之外的的脚本语言，因此我们可以让PM2和Deno搭配起来。首先安装pm2（略），然后创建一个app.sh文件，内容如下：

#!/bin/bash
deno run -A myCode.ts
1
2

保存后在当前目录执行：

pm2 start ./app.sh
1
